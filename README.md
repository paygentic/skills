# Paygentic AI Skills

Claude Code skills for merchant developers integrating Paygentic billing.

## Installation

```bash
/plugin marketplace add paygentic/ai-skills
/plugin install paygentic@paygentic-ai-skills
```

## Skills

| Skill | Purpose |
|-------|---------|
| `paygentic-explore` | Evaluate Paygentic, understand billing/metering concepts |
| `paygentic-integrate` | Write SDK integration code (TypeScript or Python) |

## Requirements

- [Claude Code](https://claude.com/claude-code) CLI
- Paygentic account with API key (get one at [app.paygentic.com](https://app.paygentic.com))
- One of: `@paygentic/sdk` (npm) or `paygentic-sdk` (PyPI)

## Updating

```bash
/plugin marketplace update paygentic-ai-skills
/plugin update paygentic@paygentic-ai-skills
```
